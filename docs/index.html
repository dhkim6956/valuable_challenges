가치도전

스스로 도전과 목표를 실천하는 것이 힘든 사람들을 위해 도전 과정을 기록하고 수행했던 도전들을 확인함으로써 성취감을 느끼게 하여 목표를 위한 생활습관 형성 뿐만 아니라 자기성장을 할 수 있도록 도와주는 앱

Main Function

  • Challenge Management

    진행 중인 도전 리스트와 진행상황을 한눈에 확인할 수 있는 인터페이스, 도전을 진행함에 따라 성장하는 귀여운 캐릭터

  • Various Challenges

    도전을 기획하기 어려운 사용자를 위한 다양한 샘플 도전 제공, 이를 바탕으로 나만의 도전을 기획하고 생성

  • Challenges History

    사용자의 도전 기록들을 기록, 관리하여 언제든 사용자가 자신의 도전 진행 히스토리를 확인할 수 있도록 구현

App Demo

도전 시작부터 인증, 관리까지

Tech Stack

  • CollectionView
    여러 업적을 화면에 표시하는데 사용
  • SearchBar
    필요한 도전을 찾는데 사용
  • CircularProgressBar
    도전 진행상황을 시각화 하는데 사용
  • Picker
    사진, 색, 선택지, 날짜 등을 선택하는데 사용
  • FSCalendarCustom
    캘린더를 통해 목표 진행상황을 나타내는데 사용
  • FileManager
    사용자가 앱을 사용하며 생성한 사진, 도전기록 등을 저장하는데 사용
